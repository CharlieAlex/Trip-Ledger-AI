# 啟動應用程式

啟動 Trip Ledger AI 的 Streamlit Web 介面。

## 快速啟動

// turbo
1. 確認依賴已安裝：
```bash
make sync
```

// turbo
2. 啟動應用程式：
```bash
make run
```

3. 在瀏覽器中開啟 http://localhost:8501

## 功能頁面

- **📤 上傳**：上傳發票照片並處理
- **📅 時間線**：按日期查看消費記錄
- **📊 分析**：消費類別統計圖表
- **🗺️ 地圖**：地理分布視覺化
- **⚙️ 設定**：API Token 與偏好設定

## 停止應用程式

在終端機按 `Ctrl+C` 停止。
